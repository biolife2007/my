# My Workspaces

This repository keeps local working materials split by direction.

## Structure

- `prolitech/` - Prolitech scripts, SQL queries, exports, and notes.
- `prolimax/` - Prolimax scripts, SQL queries, exports, and notes.
- `onebox/` - OneBox scripts, exports, integration notes, and utilities.
- `shared/` - Reusable scripts and documentation shared across directions.

Keep credentials in local `.env_*` files only. Generated temporary files should stay in `tmp/` and should not be committed.
